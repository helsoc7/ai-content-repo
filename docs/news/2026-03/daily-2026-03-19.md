### Transformers are Bayesian Networks

**Quelle:** cs.AI updates on arXiv.org
**Veröffentlicht:** 2026-03-19T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.17063

Diese Forschungsarbeit liefert eine präzise theoretische Erklärung, warum Transformer-Modelle so gut funktionieren: Sie implementieren Bayessche Netzwerke. Die Autoren zeigen auf fünf Wegen, dass jeder Sigmoid-Transformer loopy Belief Propagation auf seinem impliziten Faktorgraphen ausführt – eine Runde pro Schicht, unabhängig von den Gewichten.

**Warum relevant?** Diese Erkenntnis könnte das Verständnis und die Weiterentwicklung von LLMs grundlegend verändern, indem sie das theoretische Fundament für Transformer-Architekturen legt.

### Graph-Native Cognitive Memory for AI Agents

**Quelle:** cs.AI updates on arXiv.org
**Veröffentlicht:** 2026-03-19T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.17244

Kumiho ist eine graphbasierte kognitive Gedächtnisarchitektur für KI-Agenten, die auf formaler Belief-Revision-Semantik basiert. Sie nutzt unveränderliche Revisionen, veränderliche Tag-Pointer, typisierte Abhängigkeitskanten und URI-basierte Adressierung – Strukturen, die sowohl für kognitives Gedächtnis als auch für das Verwalten von Agentenzustand geeignet sind.

**Warum relevant?** Zuverlässiges Langzeitgedächtnis ist eine der größten offenen Herausforderungen für autonome KI-Agenten und ein wichtiger Baustein für Multi-Agenten-Systeme.

### CRAFT: Contrastive Reasoning Alignment für robuste LLMs

**Quelle:** cs.AI updates on arXiv.org
**Veröffentlicht:** 2026-03-19T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.17305

CRAFT ist ein Red-Teaming-Alignment-Framework, das die Reasoning-Fähigkeiten von Modellen und ihre internen Repräsentationen nutzt, um die Robustheit gegenüber Jailbreak-Angriffen zu verbessern. Anders als bisherige Verteidigungsansätze, die hauptsächlich auf der Ausgabeebene arbeiten, optimiert CRAFT Ziele direkt im Hidden-State-Raum.

**Warum relevant?** Die Sicherheit großer Sprachmodelle gegen Manipulation ist eine der zentralen Herausforderungen für den verantwortungsvollen KI-Einsatz.

**Tags:** llm, safety, research

### Multi-Agent Reinforcement Learning für dynamische Preisgestaltung

**Quelle:** cs.LG updates on arXiv.org
**Veröffentlicht:** 2026-03-19T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.16888

Diese Arbeit präsentiert eine systematische empirische Auswertung von Multi-Agenten-RL-Ansätzen (MAPPO und MADDPG) für die dynamische Preisoptimierung im Wettbewerb. Mithilfe einer simulierten Marktumgebung auf Basis realer Einzelhandelsdaten werden die Algorithmen unter Gesichtspunkten von Rentabilität, Stabilität und Fairness verglichen.

**Warum relevant?** Autonome Preisgestaltungs-Agenten im Einzelhandel könnten den Wettbewerb fundamental verändern und werfen gleichzeitig regulatorische Fragen auf.
