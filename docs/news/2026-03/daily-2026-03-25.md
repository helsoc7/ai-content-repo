### Session Risk Memory (SRM): Temporal Authorization for Deterministic Pre-Execution Safety Gates

**Quelle:** cs.AI updates on arXiv.org
**Veröffentlicht:** 2026-03-25T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.22350

Das Paper stellt Session Risk Memory (SRM) vor – ein leichtgewichtiges, deterministisches Modul, das Sicherheitsgates für KI-Agenten um eine zeitliche Dimension erweitert. Bisherige Systeme bewerten Aktionen nur einzeln und sind blind gegenüber Angriffen, die schädliche Absichten über mehrere scheinbar harmlose Schritte verteilen.

**Warum relevant?** Mit zunehmender Autonomie von KI-Agenten werden mehrstufige Angriffsmuster zu einem realen Sicherheitsproblem – SRM bietet einen praktischen Ansatz zur Abwehr.

### Between the Layers Lies the Truth: Uncertainty Estimation in LLMs Using Intra-Layer Local Information Scores

**Quelle:** cs.LG updates on arXiv.org
**Veröffentlicht:** 2026-03-25T04:00:00+00:00
**Link:** https://arxiv.org/abs/2603.22299

Das Paper präsentiert eine kompakte Methode zur Unsicherheitsschätzung in LLMs: Durch Analyse schichtübergreifender Übereinstimmungsmuster in einem einzigen Forward-Pass wird erkannt, wann ein Modell mit hoher Wahrscheinlichkeit falsche Antworten produziert (Halluzinationen). Die Methode erzielt vergleichbare Genauigkeit wie aufwändigere Probing-Ansätze.

**Warum relevant?** Zuverlässige Unsicherheitsschätzung ist eine Grundvoraussetzung für den vertrauenswürdigen Einsatz von LLMs in kritischen Anwendungen.

**Tags:** llm, research, safety
